# JavaCrud

## About
*Project that simulates an anime store system, where we register anime, producers, number of episodes, being able to search by id, search all, delete and update anime or producer*

## Functions

**Producer**

* *Delet producer* 
* *Get producer*
* *Update producer*
* *Save new producer*


**Anime**
* *Delet anime*
* *Get anime*
* *Update anime*
* *Save new anime*
